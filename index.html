Política de Privacidad

Última actualización: 28 de junio de 2026

Esta Política de Privacidad describe cómo Mi Quincena ("la aplicación", "nosotros") recopila, usa y protege la información de los usuarios.

1. Información que recopilamos

Mi Quincena es una aplicación de finanzas personales. La información que maneja la app incluye:

  • Datos financieros que el usuario ingresa manualmente: gastos, ingresos, metas de ahorro y deudas.
  • Configuración de la app: sueldo, presupuestos por categoría y preferencias.
  • Datos de uso de anuncios a través de Google AdMob (ver sección 4).

Todos los datos financieros se almacenan únicamente en el dispositivo del usuario. No enviamos ni almacenamos tus datos financieros en servidores externos.

2. Cómo usamos la información

La información ingresada por el usuario se usa exclusivamente para:

  • Mostrar resúmenes, estadísticas y comparaciones de gastos dentro de la app.
  • Calcular el progreso hacia metas de ahorro.
  • Enviar notificaciones locales de recordatorio (si el usuario las activa).
  • Realizar copias de seguridad locales compartidas por el propio usuario.

3. Almacenamiento de datos

Mi Quincena almacena todos los datos financieros del usuario localmente en el dispositivo usando Hive, una base de datos local. Estos datos no se sincronizan ni se envían a ningún servidor. Si el usuario desinstala la aplicación, los datos se eliminan permanentemente.

4. Google AdMob y publicidad

La versión gratuita de Mi Quincena utiliza Google AdMob para mostrar anuncios. AdMob puede recopilar información del dispositivo para personalizar anuncios, incluyendo:

  • Identificador de publicidad del dispositivo (Advertising ID).
  • Dirección IP y datos de ubicación aproximada.
  • Información sobre interacciones con anuncios.

Para más información sobre cómo Google usa esta información, consulta la Política de Privacidad de Google.

Los usuarios con Plan Pro no ven anuncios y AdMob no recopila datos de publicidad para ellos.

5. Compras dentro de la app

Mi Quincena ofrece un Plan Pro mediante compras dentro de la app procesadas por Google Play. Las transacciones de pago son manejadas exclusivamente por Google y están sujetas a su política de privacidad. No almacenamos información de tarjetas de crédito ni datos de pago.

6. Notificaciones

La app puede enviar notificaciones locales al dispositivo si el usuario las activa en la configuración. Estas notificaciones se generan localmente y no implican el envío de datos a servidores externos.

7. Autenticación biométrica

Si el usuario activa la opción de Face ID o huella dactilar, la autenticación se realiza localmente en el dispositivo usando las APIs del sistema operativo. No almacenamos datos biométricos.

8. Permisos de la aplicación

Mi Quincena puede solicitar los siguientes permisos:

  • Notificaciones: para enviar recordatorios financieros.
  • Biometría: para proteger el acceso con Face ID o huella.
  • Almacenamiento: para exportar el archivo de respaldo.
  • Internet: para mostrar anuncios de AdMob y procesar compras.

9. Privacidad de menores

Mi Quincena no está dirigida a menores de 13 años. No recopilamos intencionalmente información personal de menores de edad.

10. Cambios a esta política

Podemos actualizar esta Política de Privacidad ocasionalmente. Notificaremos cualquier cambio publicando la nueva versión en esta página con la fecha de actualización.
